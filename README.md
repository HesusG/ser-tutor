# Ser tutor

Un libro breve y concreto sobre el oficio de acompañar a quien aprende.

🔗 **Sitio:** https://hesusg.github.io/ser-tutor/

Ocho capítulos: la tutoría, la acción tutorial, el perfil del tutor, las habilidades
socioemocionales, la ética profesional, la gestión escolar, del marco al proyecto, y el cierre.
Cada capítulo tiene su propio diseño y se apoya en fuentes verificadas (referencias al pie).

Hecho como trabajo del curso *Temas Selectos de Educación* (UPAEP). Estética de libro,
neobrutalista sobria. Sin dependencias: HTML + CSS + JS estáticos.
